This is a Social media app Focused on mordern parents!!!!!

Here you can upload an image in your post, edit it before posting,
add filter to your image, etc.

It's somewhat like instagram you upload to a public feed and like 
or upvote their image.

The software is build on ionic 4 so that it could be run on most 
of the devices Android, IOS, Windows Desktop.

For javascript Angular 6 is used, along with firebase as database
and uploadCare to store and hadle images.
